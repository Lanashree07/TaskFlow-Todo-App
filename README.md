# TaskFlow - 📝 To-Do List Application
TaskFlow is an interactive To-Do List application developed using HTML, CSS, and JavaScript that provides full CRUD functionality, dynamic DOM manipulation, event handling, task filtering (All, Active, Completed), and Local Storage persistence to retain user data across browser sessions.

## Features
- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Filter Tasks (All, Active, Completed)
- Local Storage Persistence
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Local Storage API
